# Warthin-Tumor-Clinical-Imaging-Dataset
Warthin Tumor Clinical–Imaging Dataset
Introduction

This repository contains a structured research dataset focused on the clinicopathological and radiological characterization of Warthin Tumor (WT). The dataset was developed to support quantitative analysis of epidemiological, metabolic, diffusion-weighted imaging, and molecular parameters associated with WT.

The dataset includes demographic variables, smoking exposure metrics, metabolic indicators, diffusion-weighted MRI values, PET SUVmax measurements, dynamic contrast-enhancement patterns, and selected molecular markers. Comparative cases of pleomorphic adenoma and mucoepidermoid carcinoma are included to allow differential diagnostic modeling and threshold validation.

This dataset is intended for academic research, statistical modeling, and imaging biomarker evaluation in studies investigating benign and malignant parotid gland tumors.
